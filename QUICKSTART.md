# NeonTube Quick Start Guide

## 3 Easy Ways to Install

### Option 1: One-Line Install (Linux/Mac) - Fastest!
```bash
curl -sSL https://raw.githubusercontent.com/contactmukundthiru-cyber/Multi-Platform-Downloader/main/install.sh | bash
```
Then run: `neontube`

---

### Option 2: Download & Double-Click (All Platforms)

1. **Download**: Click the green "Code" button > "Download ZIP"
2. **Extract**: Unzip the downloaded file
3. **Run installer**:
   - **Windows**: Double-click `install.bat`
   - **Mac/Linux**: Double-click `install.sh` (or run in terminal)
4. **Launch**: Use the desktop shortcut or run `neontube`

---

### Option 3: Git Clone (Developers)
```bash
git clone https://github.com/contactmukundthiru-cyber/Multi-Platform-Downloader.git
cd Multi-Platform-Downloader
./install.sh
```

---

## How to Use

1. **Paste URL**: Copy any video URL and paste it in the app
2. **Choose Format**: Select Video or Audio, then pick quality
3. **Download**: Click the big blue DOWNLOAD button!

## Supported Sites

YouTube, TikTok, Instagram, Twitter/X, Facebook, Vimeo, Reddit, Twitch, SoundCloud, Spotify, and **1000+ more!**
